File: server/api/goals.ts

Schema: `CreateGoalSchema` (zod) — validiert Body für POST (userId, name, target, saved, dueDate)

Exported handler: default async (event)

Funktionen / Verhalten:
- Handler(event):
  - `GET`: Validiert `userId` → ruft `GoalService.getGoalsByUserId(userId)` → gibt Liste zurück
  - `POST`: Liest Body, validiert → baut `payload` (inkl. saved defaulting to 0) → ruft `GoalService.createGoal(payload)` → gibt Ergebnis zurück
  - Fehler/405 entsprechend

Aufruft / Abhängigkeiten:
- `GoalService.getGoalsByUserId(Number(userId))`
- `GoalService.createGoal(payload)`

Rückgabewerte:
- GET: Array von Goals
- POST: Objekt { message, goal }
