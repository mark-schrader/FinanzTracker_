== Controller (Kommunikation mit Frontend)

Controller: "Budgetcontroller" (nicht umgesetzt)

=== Controller: categories
    * erhalten der Kategorien vom Service
    * senden der Kategorien an das Frontend
    * erhalten einer neuen Kategorie vom Frontend
    * senden der neuen Kategorie an den Service
    * Fehlermeldung an das Frontend senden

=== Controller: expenses
    * erhalten der Ausgaben vom Service
    * senden der Ausgaben an das Frontend
    * erhalten einer neuen Ausgabe vom Frontend
    * senden der neuen Ausgabe an den Service
    * Fehlermeldung an das Frontend senden

=== Controller: goals
    * erhalten der Challenge vom Service
    * senden der Challenge an das Frontend
    * erhalten eines neuen Challenge vom Frontend
    * senden des neuen Challenge an den Service
    * Fehlermeldung an das Frontend senden

=== Controller: incomes
    * erhalten der Einnahmen vom Service
    * senden der Einnahmen an das Frontend
    * erhalten einer neuen Einnahme vom Frontend
    * senden der neuen Einnahme an den Service
    * Fehlermeldung an das Frontend senden

=== Controller: transaction

    * senden der Transaktionen an das Frontend
