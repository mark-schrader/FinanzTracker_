== Controller (Kommunikation mit Frontend)

=== Controller: categories

    * erhalten der Kategorien vom Service und ausgeben an das Frontend

== Service (Logik)

Service: BudgetService (nicht umgesetzt)

=== Service: CategoryService
    * erhalten der Kategorien vom Repository
    * senden der Kategorien an den Controller
    * erhalten einer neuen Kategorie vom Controller
    * senden der neuen Kategorie an das Repository
    * ID vom Controller erhalten
    * je nach wünsch löschen/aktualisieren der Kategorie im Repository
